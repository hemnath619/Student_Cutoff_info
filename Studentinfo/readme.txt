Follow the below steps,

1. Copy the client and server to local folders,
2. Open them individually in cmd prmpt then type npm install (it downloads required files fro the project),
3. Provide the details in the .env file in server folder so that the mongodb can be connected with the project and mails can be sent,
4. Finally type npm start and run the project and provide the inputs in the form and click submit, your data will be stored in MongoDb and a mail will be sent to the user specified mail id.
